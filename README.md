# Solviem Landing Page

Landing page corporativa para **Solviem Consultoría Contable y Empresarial**, desarrollada con HTML y CSS puro.

El objetivo de este proyecto es presentar los servicios de Solviem de forma profesional, clara y accesible para freelancers, emprendedores y pequeñas empresas que buscan apoyo en contabilidad, impuestos y administración financiera.

---

## 🚀 Descripción

Solviem es una firma enfocada en brindar soluciones contables, fiscales y financieras para personas físicas y negocios en México.

La landing page busca transmitir confianza, profesionalismo y simplicidad, siguiendo la filosofía:

> **"Tu paz financiera comienza aquí."**

---

## ✨ Características

* Diseño moderno y responsivo.
* Navegación de una sola página (Single Page Landing).
* Identidad visual basada en la marca Solviem.
* Sección Hero con propuesta de valor.
* Servicios principales.
* Proceso de trabajo.
* Llamado a la acción (CTA).
* Favicon personalizado.
* Preparado para despliegue en GitHub Pages.
